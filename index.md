# Privacy Policy — Track Me

Last updated: 2026-07-08

Track Me is a local time-tracking Chrome extension.

## What data is processed
- Tasks, session timestamps, and extension settings.
- Optional backup files generated by the user or automatic backup setting.

## Where data is stored
- Data is stored locally in the browser using Chrome Extension storage (`chrome.storage.local`).
- Backup files are saved to the user's local Downloads folder (or user-selected location for manual backup).

## Data sharing
- Track Me does not send personal data to external servers.
- Track Me does not sell, transfer, or share user data with third parties.

## Permissions used
- `storage` — to save tasks and settings locally.
- `downloads` — to create backup files on the user's device.
- `alarms` — to run optional hourly automatic backups when enabled.

## User control
- Users can delete tasks and local data directly in the extension.
- Users can enable/disable automatic backup in Settings.
- Users can uninstall the extension at any time.
